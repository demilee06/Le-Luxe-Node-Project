The API Routes

Home

HTTP Methods Path Description
GET / Home page

Users

HTTP Methods Path Description
GET /users Gets all the users on the table
GET /users/:id Gets a single user by using their userID
POST /users/register Inserts a single user
PATCH/PUT /users/update/:id Update user details by ID
DELETE /users/delete/:id Delete a user by ID

Products

HTTP Methods Path Description
GET /product Gets all the products from the table
GET /product/:id Gets a single product by using the prodID
POST /product/add Adds one product
PATCH /product/update/:id Edits a single product by using the prodID
DELETE /product/delete/:id deletes a single product by using the prodID
